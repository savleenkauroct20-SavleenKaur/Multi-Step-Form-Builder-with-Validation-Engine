🧭 Introduction
The Smart Multi-Step Form is a front-end web application designed to streamline data collection through a structured, user-friendly interface. Instead of presenting users with a long and complex form, the application divides input into multiple logical steps, improving usability and reducing cognitive load.
The system incorporates conditional rendering, validation mechanisms, and modern UI enhancements to deliver an efficient and interactive form-filling experience.

🎯 Objective
The primary objective of this project is to design and develop a dynamic and interactive form system that:
Guides users through a step-by-step data entry process
Validates user input to ensure data accuracy
Dynamically displays fields based on user responses
Enhances user experience through clean UI and smooth interactions

⚙️ System Workflow
The user begins at the Personal Information step
Navigates to the Education section
Conditional fields are displayed based on responses (e.g., student status)
Optional file inputs allow document submission
A progress indicator provides real-time completion feedback
Upon submission, a confirmation screen displays collected data
The user may reset the form using the Start Over functionality

🚀 Key Features
🔹 Multi-Step Navigation
Segments the form into manageable steps
Supports forward and backward navigation
🔹 Conditional Logic
Dynamically renders fields based on user input
Ensures only relevant information is requested
🔹 Validation Mechanism
Implements required field checks
Validates email format using regex
Displays inline error messages for better UX
🔹 Progress Indicator
Displays current step and completion status
Improves transparency and user engagement
🔹 Optional File Upload
Allows users to upload 10th and 12th certificates
File submission is optional and non-blocking
🔹 Theme Support (Light/Dark Mode)
Provides toggle between light and dark themes
Enhances accessibility and user preference
🔹 Smooth UI Transitions
Implements subtle animations between steps
Improves visual continuity
🔹 Success Screen
Displays a structured summary of submitted data
Provides clear confirmation of successful submission
🔹 Form Reset Functionality
Enables users to restart the form from the initial state
Clears all stored data
